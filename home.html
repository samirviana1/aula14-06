Git é o sistema de controle de versão open source mais usado no mundo atualmente! Ele é usado para controlar
o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software. Ele
permite mais flexibilidade no fluxo de trabalho, segurança e desempenho.

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele
permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos
privados e/ou Open Source de qualquer lugar do mundo. GitHub é amplamente utilizado por programadores para
divulgação de seus trabalhos ou para que outros programadores contribuam com o projeto, além de promover
fácil comunicação através de recursos que relatam problemas ou mesclam repositórios remotos (issues, pull request).
